# Dashboard đề xuất

## 1. Sales Overview
KPI: Doanh thu, Tổng đơn, Số sản phẩm, AOV. Chart: doanh thu theo tháng, top danh mục, top seller.

## 2. Payment Overview
KPI: Tổng thanh toán, tỷ lệ payment type, nhóm trả góp. Chart: stacked column payment type theo tháng.

## 3. Review Overview
KPI: Điểm đánh giá TB, tỷ lệ có đánh giá, số đơn chưa đánh giá. Chart: stacked column đơn có/chưa review.

## 4. Delivery SLA
KPI: tỷ lệ đúng/sớm hạn, tỷ lệ trễ hạn, số ngày giao TB. Chart: stacked column Đúng/Sớm/Trễ theo tháng, scatter số ngày giao vs điểm review.

## 5. NoSQL Supply Sales
KPI: doanh thu supply, số sale_id, top product. Chart: revenue by storeLocation, purchaseMethod.
